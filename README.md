# Perform AI Fullstack Assessment

Small full-stack prototype. A NestJS backend exposes a single asynchronous `POST /analysis` endpoint that simulates biomechanics processing, and an Ionic + Angular frontend submits an athlete name and polls for results.

## Layout

```
backend/   NestJS API (TypeScript)
frontend/  Ionic + Angular app
docs/      Architecture note
```
